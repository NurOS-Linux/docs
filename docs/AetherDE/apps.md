# Aether Apps

Aether Apps (ранее Delta Apps) — это набор программ (компонентов) для AetherDE, образующих полноценную экосистему десктопных приложений.

## Основные приложения

### Утилиты и продуктивность

| № | Название | Описание |
|---|----------|----------|
| 1 | **Æther Math** | Калькулятор |
| 2 | **Æther Scriptum** | Текстовый редактор с поддержкой LSP и Markdown |
| 3 | **Æther Astrum** | Файловый менеджер |
| 4 | **Æther Focus** | Заметки (часть Scriptum) |
| 5 | **Æther Tikra** | Часы |
| 6 | **Æther Tyndra** | Календарь |

### Системные инструменты

| № | Название | Описание |
|---|----------|----------|
| 7 | **Æther Tunekit** | Кастомизация DE, GTK, Qt и других тем |
| 8 | **Æther Setkit** | Настройки дистрибутива и init системы |
| 9 | **Æther Vigil** | Монитор ресурсов системы |
| 10 | **Æther Kiro** | Эмулятор терминала |
| 11 | **Æther ATS** | About This System - информация о системе |

### Мультимедиа

| № | Название | Описание |
|---|----------|----------|
| 12 | **Æther Lucida** | Просмотрщик фото и камера |
| 13 | **Æther Sonora** | Аудиоплеер |
| 14 | **Æther Visio** | Видеоплеер |
| 15 | **Æther Captus** | Скриншоты и запись экрана |

### Приветствие и магазин

| № | Название | Описание |
|---|----------|----------|
| 16 | **NurOS Hello** | Приветственный экран для новых пользователей |
| 17 | **Æther Tourist** | Магазин приложений |

### Дополнительные приложения

| № | Название | Описание |
|---|----------|----------|
| 18 | **Æther Streets** | Карты и навигация |

## Языки программирования

Приложения Aether разрабатываются с использованием:

- **Vala** — основной язык для GTK приложений
- **C** — системные компоненты и библиотеки
- **C++** — производительные компоненты

## Технологии и фреймворки

### UI Frameworks

- **GTK 4** — современный toolkit для пользовательского интерфейса
- **Qt 6** — кроссплатформенный фреймворк (для некоторых приложений)
- **libadwaita** — компоненты GNOME для адаптивного UI

### Дополнительные технологии

- **GStreamer** — обработка мультимедиа (Sonora, Visio)
- **WebKitGTK** — для веб-контента (Tourist, Streets)
- **LSP** (Language Server Protocol) — умная подсветка кода (Scriptum)
