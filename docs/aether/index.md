# AetherDE

**AetherDE** — настольная среда для Linux, построенная на принципах минимализма, производительности и удобства использования.

## Основные компоненты

| Компонент | Название | Описание |
|-----------|----------|----------|
| **Desktop Environment** | AetherDE | Основная настольная среда |
| **Window Manager** | NexameWM | Композитный оконный менеджер |
| **Display Manager** | Halo | Менеджер входа в систему |
| **GUI Framework** | GTK + Qt | Гибридный подход |
| **Design Guidelines** | GNOME HIG | Принципы проектирования интерфейса |
| **Тема оформления** | Bluesweet | Единый визуальный стиль |

## Принципы дизайна

AetherDE реализует спецификацию GNOME HIG и опирается на четыре ключевых принципа:

1. **Простота** — минимализм в интерфейсе, без лишнего шума
2. **Последовательность** — единообразие элементов во всех приложениях
3. **Доступность** — поддержка пользователей с ограниченными возможностями
4. **Отклик** — быстрая реакция на действия пользователя

## Технологии

- **Vala** — основной язык разработки GTK-приложений
- **C / C++** — системные компоненты и производительные части
- **GTK 4 + libadwaita** — UI-фреймворк и адаптивные компоненты
- **Qt 6** — для отдельных приложений

## Состав экосистемы

AetherDE включает полный набор приложений — [Aether Apps](./apps.md), покрывающих повседневные задачи: файловый менеджер, текстовый редактор, терминал, мультимедиа, настройки системы и магазин приложений.
